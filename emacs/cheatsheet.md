# Emacs initial commands

C-x C-f: Open file
C-x C-s: Save file
C-x b: Switch to other buffer
C-x k: Kill buffer

C-x o: Switch to next window
C-x 3: Split window side-by-side
C-k: Kill to end of line
C-x u: Undo
C-o: Insert line
C-x g: cancel

M-x load-file: load file
C-x C-e: evaluate sexp

M-x Term
